# Starting Guild #

### Description ###
------------------------------------------------------------------------------------------------------------------
This module auto-assigns new players to specific guilds.

- New characters are added to a specified guild on first login.
- Config: Set ID for specific Horde and Alliance guilds


### Data ###
------------------------------------------------------------------------------------------------------------------
- Type: Player/Server
- Script: StartGuild
- Config: Yes
- SQL: Yes
